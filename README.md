# commandant-rules-core
MPL-2.0 ruleset collection for commandant. Original rulesets for common shell tools, LLM-generated and human-reviewed. Improvements to ruleset files flow back; no obligations on your application code.

> **WARNING**: These rules are NOT ready for any real use. Don't use them.
